Spec Text

A node node is editable if the following algorithm returns true:

  1. If node is not an instance of Element or Attr, then return false.
  2. If node is an editing host, then return false.
  3. If node is an Element and node's contenteditable attribute is set to the false state, then return false.
  4. If var is an instance of Attr, let parent be the node's ownerElement. Otherwise, if node is an instance of Element, let parent be the node's parentElement.
  5. If parent is null, then return false.
  6. If parent is neither an editing host nor editable, then return false.
  7. If node is an Element in the HTML namespace, SVG namespace, or MathML namespace, then return true.
  8. If node is an Attr and parent is an element in the HTML namespace, then return true.
  9. Return false.
